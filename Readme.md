# Разрабтка аналитичекой системы для начинающих инвесторов (ВКР)

## Описание проекта

Интерактивная аналитическая система, разработанная специально для начинающих инвесторов. Платформа позволяет пользователям отслеживать свои инвестиционные портфели, получать автоматизированную аналитику от ИИ, визуализировать динамику счёта и оставаться в курсе последних финансовых новостей.

Система объединяет 4 ключевых компонента:

- Персональный кабинет для управления инвестиционным кошельком

- AI-аналитик для генерации текстовой аналитики на основе вашего портфеля

- Графические инструменты для наглядного представления данных

- Новостная лента с актуальной финансовой информацией

## Пользователи
Проект ориентирован на:

- Начинающих инвесторов, которые только знакомятся с финансовыми рынками

- Студентов экономических и финансовых специальностей

- Финансовых консультантов, работающих с клиентами-новичками

## Возможности
### Управление инвестиционными кошельками: 

- Внесение данных о ценных бумагах и других активах

- Отслеживание стоимости портфеля

- История операций и пополнений

### ИИ-аналитика:

- Автоматическая генерация текстовых отчётов о состоянии портфеля

- Персонализированные рекомендации на основе вашего профиля риска

- Анализ диверсификации и распределения активов

- Сигналы о потенциальных рисках и возможностях

### Графическая аналитика:
- Интерактивные графики динамики портфеля

- Визуализация распределения активов (pie-чарты, heatmaps)

### Финансовые новости:

- Учёт последних событий при аналитике

- Отображение актуальных для пользователя новостей


## Доплнительный контекст

- Приложение разрабатывается только для российских инвестиционных платформ

- Приложение работает только с рублёвыми активами или активами, имеющими рублёвый эквивалент
